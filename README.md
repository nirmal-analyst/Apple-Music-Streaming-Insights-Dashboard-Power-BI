Description:
Designed and developed a visually engaging Power BI dashboard inspired by Apple Music’s interface to analyze music streaming trends. The dashboard provides key insights into artist performance, song popularity, and user preferences across different years and album types.

Key Features:

KPIs: Displays total distinct songs, artists, average popularity score, and average song duration.

Artist Insights: Highlights top-performing artists such as Taylor Swift, Drake, and Eminem with total song counts.

Song Trends: Visualized most-streamed songs and year-wise song distribution (2023 vs 2024).

Categorical Analysis: Breakdown by song type (single/album) and explicit vs non-explicit content.

Monthly Trends: Showcases monthly patterns in distinct song releases and average popularity.

Apple-Inspired UI: Customized theme using Apple’s dark mode aesthetic with clean gradients and rounded visuals for a modern look.

Tools & Techniques:
Power BI, DAX, Power Query, Data Modeling, Custom KPI Cards, Interactive Filters
